Formatos @font-face

String FontFormat Common extensions
"woff" WOFF (Web Open Font Format) ".woff"
"truetype" TrueType ".ttf"
"opentype" OpenType ".ttf , otf"
"embedded-opentype" Embedded-Opentype ".eot"
"svg" SVG Font ".svg , .svgz"

@font-face : Permite vincular fuentes sin necesidad de que el cliente
las tenga que instalar en el ordenador

@font-face {
font-family: VariableParaRelacion;
/* 2 formatos en el servidor por si alguno de ellos falla */
src: url(FiraMono-Regular.woff) format("woff"),
url(FiraMono-Regular.eot) format("embedded-opentype");
}

section, table, tr, th, td {
font-family: VariableParaRelacion;
color: black;
}
